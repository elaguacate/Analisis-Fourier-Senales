# Análisis de Señales en el Dominio del Tiempo y Frecuencia

Este proyecto analiza señales utilizando Python y la Transformada de Fourier.

## Señales analizadas
- Senoidal
- Pulso rectangular
- Escalón

## Conceptos aplicados
- FFT
- Magnitud y fase
- Linealidad
- Desplazamiento temporal
